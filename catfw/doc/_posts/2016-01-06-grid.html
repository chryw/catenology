The grid system is based on 12 columns with a max width of 960px. .container class is the main wrapper of the grid system. To use the grid system, wrap content with .row class first, then apply column classes to the children.

<div class="container">
	<div class="row">
		<div class="four columns">four</div>
		<div class="eight columns">eight</div>
	</div>
</div>
